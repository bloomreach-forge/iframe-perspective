Release Notes

2.1.0

  • Compatible with Hippo CMS 10

1.03.00

  • Upgrade to be compatible with 7.9.
  • Support IFrame related security headers (X-Frame-Options, Content-Security-Policy, X-Content-Security-Policy, X-Webkit-CSP).

1.02.00

  • Supports IFrame Resizing when the window resizes.
  • This version is compatible with Hippo CMS 2.21.06 or later.

1.01.00

  • Initial release.
  • This version is compatible with Hippo CMS 2.19.xx or [2.21.00, 2.21.06).
