Building from source

Requirements

  • Java 7 or higher SDK

  • Maven 3

  • Subversion client

Checkout the source code

Check out the trunk source code from the SVN repository: 

svn co http://forge.onehippo.org/svn/iframeprspectiv/iframeprspectiv/trunk

Build the project using Maven: 

mvn install

Build all the reports and the site (requires Maven 2): 

mvn site
